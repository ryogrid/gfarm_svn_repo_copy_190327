gfdump

Name

gfdump(8) -- Dump and restore Gfarm file system metadata

Synopsis

gfdump -d [-v] [-f filename]

gfdump -r [-v] [-f filename]

DESCRIPTION

gfdump dumps or restores Gfarm file system metadata including directory information, file information, and replica copy information.

gfdump does not dump and restore file system node information in Gfarm metadata. That is why gfhost -M and gfhost -R are required to dump and restore the complete file system metadata in Gfarm file system, respectively.

Note that gfdump accesses to the Gfarm metadata server directly regardless of setting of a Gfarm metadata cache server, it is needed to be executed on a client host that can access to the Gfarm metadata server.

OPTIONS

-d

Dump the Gfarm file system metadata.

-r

Restore Gfarm file system metadata.

-f filename

Dump to the specified filename, or restore from the specified filename. When -f is not specified, stdout or stdin is assumed.

-v

display verbose output.
