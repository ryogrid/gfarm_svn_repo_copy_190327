gfxattr

Name

gfxattr -- Gfarm ファイルシステムのファイルの拡張属性を操作する

Synopsis

gfxattr [-s | -g | -r] [-f xattrfile] file

DESCRIPTION

指定したファイルの拡張属性を操作します． 拡張属性は任意の文字列です．特に，アプリケーションデータのメタデータ としてのXML文書などが想定されています． -sオプションを指定すると、拡張属性を書込みます． -gオプションを指定すると、拡張属性を読込みます． -rオプションを指定すると、拡張属性を消去します． -s，-g，-rオプションは必ず何れかが指定される必要があります．

OPTIONS

-s

拡張属性を設定します． すでに拡張属性が設定されている場合は上書きされます．

-g

拡張属性を表示します．

-r

拡張属性を消去します．

-f xattrfile

-sオプションと共に用いると，指定されたファイルから拡張属性を読込みます． このオプションが指定されない場合は，標準入力から拡張属性を読み込みます． -gオプションと共に用いると，指定されたファイルに拡張属性を書込みます． このオプションが指定されない場合は，標準出力に拡張属性を書込みます．

-?

引数オプションを表示します．
