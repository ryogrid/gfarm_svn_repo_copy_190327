gfdump

Name

gfdump(8) -- Gfarmメタデータをダンプ・リストアする

Synopsis

gfdump -d [-v] [-f filename]

gfdump -r [-v] [-f filename]

DESCRIPTION

Gfarm ファイルシステムのディレクトリ情報，ファイル情報， 複製情報などのメタデータをダンプ・リストアします．

gfdump がダンプ・リストアするメタデータにはファイルシステム・ ノードの情報は含まれません．そのため，完全にメタデータのダンプ・ リストアを行うためには，gfhost -M あるいは gfhost -R により別途 ファイルシステム・ノード情報をダンプ・リストアする必要があります．

なお，gfdump は，メタデータ・キャッシュサーバではなくメタデータ・ サーバを直接参照するため，メタデータ・サーバにアクセス可能なホスト で実行する必要があります．

OPTIONS

-d

Gfarm ファイルシステム・メタデータのダンプを行います．

-r

Gfarm ファイルシステム・メタデータのリストアを行います．

-f filename

指定された filename にダンプ，あるいは指定された filename からリストアを行います．-fオプションが指定されない 場合は，標準入出力が利用されます．

-v

詳細な情報を表示します．
