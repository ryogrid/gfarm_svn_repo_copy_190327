gfsck

Name

gfsck(8) -- Gfarmファイルシステムのメタデータを修復する

Synopsis

gfsck [-v] [-f] [-h hostname]

DESCRIPTION

gfsck は Gfarm ファイルシステムの指定されたファイル， あるいは指定されたディレクトリ以下の全ファイルを対象に， ファイル複製を保持する全ホストに対し， 対応する実際のファイルが存在するかどうか調べます． ファイルが存在しない場合はそのメタデータを消去し， メタデータと実際のファイルの間の一貫性を保ちます． Gfarm ファイルシステムに対する通常のファイルアクセス時でも， 同様のメタデータの修復は自動的に行われますが， gfsck は指定された全ファイル， 全ファイル複製保持ホストに対して検査を行います．

なお，gfsck は実ファイルの存在を検査しますが， メタデータに登録されているファイルかどうかの検査は行いません． その検査が必要な場合は gfsplck を用います．

なお，gfsck は，メタデータ・キャッシュサーバではなくメタデータ・ サーバを直接参照するため，メタデータ・サーバにアクセス可能なホスト で実行する必要があります．

OPTIONS

-f

実際のファイルが存在しない時だけではなく， ホストが落ちている場合もメタデータを消去します．

-h hostname

検査するホストを hostname で指定されたホストに限ります．

-v

詳細な情報を表示します．
